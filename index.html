BriarMath math editor

  • Under the "Add formula" button you will find common notations used in mathematics, physics and chemistry. In addition, special characters can be used to write the formula.
  • Formulas can be built by clicking the menu entries and/or typing LaTeX.
  • In the editor\'s text field, you can write text and formulas, add images, and import .txt and .html files.
